Для запуска (Linux):
Уствновить пакет graphviz (нужен для визуализации):
sudo apt-get update
sudo apt-get install graphviz

Собрать и запустить main.cpp:
g++ -o main main.cpp 
./main

Ввод: единственное натуральное число n - число вершин
Вывод: В папке output - все неизоморфные деревья на n вершин

Алгоритм использует хэш-функцию для деревьев, вычисляющуюся за O(n), считающую одинаковый хэш для изоморфных деревьев. 
Алгоритм работает динамически:
Для 2 вершин существует лишь одно дерево:
Для k от 1 до n, зная все деревья на k-1 вершин находятся все деревья на k вершин:
К каждой вершине каждого дерева прибавляем лист, полученный граф кладем в unordered_set с указанной выше хэш функцией.
В итоге находим все возможные деревья на k вершин с точностью до изоморфизма.
